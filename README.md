# SimplePersistenceChatbot - Retrieval Augmented Chatbot
Developed a customizable AI-powered chatbot platform designed for business knowledge management and intelligent document interaction. The application leverages OpenAI's language models to provide conversational AI capabilities while allowing organizations to securely manage and query their own data sources.

## Key Features
- Document upload and ingestion pipeline for business-specific knowledge bases.
- Retrieval-Augmented Generation (RAG) workflow enabling users to ask questions directly from uploaded documents.
- Web content integration through URL ingestion and web scraping for dynamic knowledge acquisition.
- Conversational interface with persistent chat history and context-aware responses.
- Secure storage of documents and conversation records for organizational use.
- Customizable business-focused AI assistant that can be tailored to different domains and workflows.
- Integration with OpenAI APIs for natural language understanding and response generation.
